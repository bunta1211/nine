# 標準デザイン リデザイン計画（白〜グレー＋オレンジアクセント）

## 方針

- **基本配色**: 白〜グレーで統一。視覚ノイズを減らし、読みやすさと品格を優先する。
- **アクセント**: 目立たせる必要がある部分にのみオレンジ系を使用（主要ボタン・リンク・アクティブ状態・フォーカス）。
- **可読性**: すべてのテキストでコントラスト比を確保（本文 4.5:1 以上、大きい文字 3:1 以上）。

---

## カラーパレット定義

### グレースケール（背景・枠・テキスト）

| 用途 | 色 | 備考 |
|------|-----|------|
| 純白（カード・入力・ヘッダー面） | `#ffffff` | 背景の基準 |
| 背景 L1 | `#fafafa` | ページ・パネル下地 |
| 背景 L2 | `#f5f5f5` | 中央パネルなど |
| 背景 L3 | `#eeeeee` | 区切り・サブエリア |
| 枠線・区切り線 | `#e0e0e0` | ボーダー、divider |
| 薄いグレー | `#bdbdbd` | 無効・プレースホルダー境界 |
| 本文（濃） | `#1a1a1a` | 見出し・本文（白背景で 15:1 以上） |
| 本文（中） | `#424242` | サブテキスト（白背景で 10:1 以上） |
| 補助テキスト | `#616161` | キャプション・muted（白背景で 5.7:1） |
| 弱いテキスト | `#757575` | プレースホルダー・hint（4.5:1 以上） |

### オレンジアクセント（目立たせる部分のみ）

| 用途 | 色 | 備考 |
|------|-----|------|
| メインアクセント | `#e67e22` | 主要ボタン・リンク・アクティブタブ |
| アクセントホバー | `#d35400` | ホバー・フォーカス |
| アクセント上の文字 | `#ffffff` | ボタン内テキスト（コントラスト確保） |
| リンク | `#e67e22` → ホバー `#d35400` | 下線は任意 |

## 可読性チェック

- **#1a1a1a on #ffffff**: 約 16:1（AAA）
- **#424242 on #ffffff**: 約 12:1（AAA）
- **#616161 on #ffffff**: 約 5.7:1（AA）
- **#757575 on #ffffff**: 約 4.6:1（AA）
- **#ffffff on #e67e22**: 約 3.5:1（大きなボタン・タブで AA の 3:1 を満たす）

## 実装箇所

- `includes/design_config.php`
  - `DESIGN_DEFAULT_ACCENT` → `#e67e22`
  - `getDefaultDesignTokens()` → 上記パレットで全面更新
  - `getThemeConfigs()['lavender']` → 上記パレットで全面更新
- `DOCS/STANDARD_DESIGN_SPEC.md` → カラーパレット表を新仕様に更新
- `DESIGN_ASSET_VERSION` を 1 つ上げる
- **チャットメッセージ欄**: `assets/css/chat-main.css` の `.message-card` は `--dt-msg-*` と白/グレーのフォールバックのみ。`body[data-theme="lavender"]` の紫オーバーライドは削除済み。
- **アバター**: `assets/css/chat-main.css` の `.conv-avatar` / `.member-avatars .avatar` はグレー統一。`includes/chat/sidebar.php` は `avatar-grey`、`scripts.php` の `friendColors` は `['avatar-grey']`、`modals.php` は `conv-avatar avatar-grey`。
