# AWS はじめての契約・進め方ガイド

AWSアカウントをまだお持ちでない方向けの、契約から初回構築までの丁寧な進め方です。

---

## 前提：契約に必要なもの

| 項目 | 説明 |
|------|------|
| **メールアドレス** | パスワード再設定などに使用（Gmail等） |
| **クレジットカード** | 本人確認用。初回1ドル程度が一時保留される可能性あり |
| **電話番号** | SMSまたは音声で認証コードを受信 |
| **住所** | ローマ字（半角アルファベット）で入力 |

※無料枠内でも、本人確認のためカード登録は必須です。

---

## ステップ1: AWSアカウント作成

### 1-1. 登録ページへアクセス

**公式サインアップページ（日本語）**

- **https://aws.amazon.com/jp/register-flow/**  
  （アカウント作成の流れ）
- **https://portal.aws.amazon.com/billing/signup**  
  （直接サインアップ開始）

### 1-2. 登録の流れ（おおよそ15〜20分）

| 順番 | 内容 | 補足 |
|------|------|------|
| 1 | メールアドレス・AWSアカウント名 | 半角英数字でアカウント名を入力 |
| 2 | 認証コード入力 | メールに届く6桁コードを入力 |
| 3 | パスワード設定 | 強力なパスワードを設定 |
| 4 | 連絡先情報 | 個人 or ビジネス、氏名・住所・電話番号（ローマ字） |
| 5 | 支払い情報 | クレジットカードまたはデビットカードを登録 |
| 6 | 本人確認 | 電話番号へSMSまたは音声でコードが届く |

**重要**: 電話認証の際、**非通知着信拒否**を解除しておいてください。

### 1-3. アカウントの有効化

- 通常は数分〜24時間でアカウントが有効になります
- メールで案内が届くまで待ちます

## ステップ2: 無料枠の理解（2025年7月以降）

### 2-1. 公式ドキュメント

- **AWS 無料利用枠（日本語）**  
  https://aws.amazon.com/jp/free/

### 2-2. 現在の無料プラン概要

| 項目 | 内容 |
|------|------|
| **期間** | 最大6ヶ月 または クレジット消費まで |
| **初回クレジット** | 100ドル（約15,000円相当）が自動付与 |
| **追加クレジット** | アクティビティ完了で最大100ドル追加（合計最大200ドル） |
| **料金** | クレジット内であれば実質無料 |

※従来の「12ヶ月無料」から2025年7月に新しいプランに変更されています。

### 2-3. 本プロジェクトで主に使うサービス

| サービス | 無料枠例 | 用途 |
|----------|----------|------|
| EC2 | t3.micro など | Webサーバー（PHP） |
| RDS | db.t3.micro（MySQL） | データベース |
| S3 | 5GB | ファイル保存 |
| Route 53 | ホストゾーン1件 | ドメイン設定 |

## ステップ3: アカウント作成後のセキュリティ設定（必須）

### 3-1. MFA（多要素認証）の設定

ルートユーザーにMFAを設定すると、不正アクセスを防ぎやすくなります。

- **MFAの設定手順（公式）**  
  https://docs.aws.amazon.com/ja_jp/IAM/latest/UserGuide/id_credentials_mfa_enable.html

※スマートフォンの認証アプリ（Google Authenticator、Microsoft Authenticator等）を用意してください。

### 3-2. 請求アラートの設定（予算超過防止）

**必ず設定しておくことを推奨します。**

#### 方法A: 請求設定でアラートを有効化

1. **AWSマネジメントコンソール** にログイン  
   https://console.aws.amazon.com/
2. 右上のアカウント名 → **「請求ダッシュボード」**
3. 左メニュー **「設定」** → **「請求設定」**
4. **「アラート設定」** で以下にチェック
   - AWS無料利用枠アラート
   - CloudWatch請求アラート
5. 通知先メールアドレスを入力して保存

#### 方法B: 予算（Budget）を設定

1. **請求ダッシュボード** → **「予算」** → **「予算の作成」**
2. 例: 月額10,000円の予算を設定
3. 80%でアラート、100%でアラートなどを設定

- **AWS Budgets の使い方（公式）**  
  https://repost.aws/ja/knowledge-center/budgets-track-spending-and-usage

## ステップ4: 最初にやることの順序

```
1. AWSアカウント作成（ステップ1）
     ↓
2. アカウント有効化を待つ
     ↓
3. MFA設定（ステップ3-1）
     ↓
4. 請求アラート設定（ステップ3-2）
     ↓
5. 現行環境のバックアップ
   - DBダンプ
   - uploads/ フォルダ
   - config/ 設定ファイル
     ↓
6. EC2・RDS の構築（AWS_MIGRATION_STEPS.md 参照）
```

## よくある質問

### Q. クレジットカードがないと使えない？

本人確認のため、カード登録は必須です。無料枠内でも、不正利用防止の目的で登録が求められます。

### Q. 無料枠を超えたらどうなる？

請求アラートを設定しておけば、事前にメールで通知されます。予算を超えないよう、アラートを早めに設定してください。

### Q. いつでも解約できる？

はい。使っていないリソースを停止・削除すれば、それ以降の課金は発生しません。アカウント自体の閉鎖手順は AWS サポートに問い合わせます。

### Q. 日本語でサポートは受けられる？

はい。AWS サポート（有料プラン）や、各種ドキュメントは日本語で提供されています。無料枠利用中は基本的にドキュメント・コミュニティでの自己解決になります。
