# デザイン機能

## 概要

ユーザーがアプリの見た目をカスタマイズできる機能。
テーマカラー、背景、フォントサイズなどを変更可能。

---

|--------|------|
| ライト | 白基調の明るいテーマ |

## 🗄️ データベース設計

### user_ui_settings テーブル

```sql
CREATE TABLE user_ui_settings (
    id INT UNSIGNED AUTO_INCREMENT PRIMARY KEY,
    user_id INT UNSIGNED NOT NULL UNIQUE,
    
    -- カラー設定
    color_mode ENUM('light', 'dark', 'system') DEFAULT 'system',
    accent_color VARCHAR(20) DEFAULT 'green',
    
    -- 背景テーマ
    background_theme ENUM('none', 'fuji', 'snow', 'ginkgo', 'sea', 'night') DEFAULT 'none',
    
    -- フォント
    font_size ENUM('small', 'medium', 'large', 'xlarge') DEFAULT 'medium',
    
    -- パネル設定
    left_panel_width INT DEFAULT 280,
    right_panel_width INT DEFAULT 320,
    left_panel_collapsed TINYINT(1) DEFAULT 0,
    right_panel_collapsed TINYINT(1) DEFAULT 0,
    
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    
    FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;
```

## 🔗 API

| エンドポイント | メソッド | 機能 |
|---------------|:--------:|------|
| `/api/settings.php?action=get_ui` | GET | UI設定取得 |
| `/api/settings.php?action=save_ui` | POST | UI設定保存 |

### 設定保存リクエスト

```json
{
  "action": "save_ui",
  "color_mode": "dark",
  "accent_color": "purple",
  "background_theme": "fuji",
  "font_size": "medium"
}
```

## 💡 実装ポイント

### 設定の適用

```javascript
function applyUISettings(settings) {
    const root = document.documentElement;
    
    // カラーモード
    root.classList.remove('light', 'dark');
    if (settings.color_mode === 'system') {
        const prefersDark = window.matchMedia('(prefers-color-scheme: dark)').matches;
        root.classList.add(prefersDark ? 'dark' : 'light');
    } else {
        root.classList.add(settings.color_mode);
    }
    
    // アクセントカラー
    root.classList.remove('accent-blue', 'accent-purple', 'accent-orange', 'accent-red');
    if (settings.accent_color !== 'green') {
        root.classList.add('accent-' + settings.accent_color);
    }
    
    // フォントサイズ
    root.classList.remove('font-small', 'font-large', 'font-xlarge');
    if (settings.font_size !== 'medium') {
        root.classList.add('font-' + settings.font_size);
    }
    
    // 背景テーマ
    document.body.classList.remove('theme-fuji', 'theme-snow', 'theme-ginkgo');
    if (settings.background_theme !== 'none') {
        document.body.classList.add('theme-' + settings.background_theme);
    }
}
```
