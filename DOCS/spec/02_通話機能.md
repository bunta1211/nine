# 通話機能

## 概要

Jitsi Meetを利用した音声通話・ビデオ通話機能。

---

## 📞 通話の種類

| 種類 | 説明 |
|------|------|
| 音声通話 | カメラOFFの通話 |
| ビデオ通話 | カメラONの通話 |

---

## 🔧 技術仕様

| 項目 | 値 |
|------|-----|
| 使用サービス | Jitsi Meet External API |
| ドメイン | meet.jit.si（公開サーバー） |
| 認証 | 不要（ルーム名で管理） |
| 同時参加 | 最大50人 |

---

## 📱 通話開始フロー

### 1対1通話（DM）

```
1. DMの相手を選択
2. 📞（音声）または 📹（ビデオ）ボタンをクリック
3. 相手に着信通知が届く
4. 相手が応答すると通話開始
```

### グループ通話

```
1. グループを選択
2. 📞（音声）または 📹（ビデオ）ボタンをクリック
3. 参加可能な状態になる（他メンバーは任意参加）
4. 「通話中」表示がグループに表示される
```

## 🔘 通話中の操作

| ボタン | 機能 |
|:------:|------|
| 🎤 | マイクのON/OFF |
| 📹 | カメラのON/OFF |
| 🖥️ | 画面共有 |
| 💬 | 通話内チャット |
| 📞 | 通話終了 |

## 🔔 着信通知

### 着信時の表示

```
┌─────────────────────────────────────────┐
│                                         │
│        📞 田中太郎からの着信            │
│                                         │
│     [応答]          [拒否]              │
│                                         │
└─────────────────────────────────────────┘
```

- 着信音が鳴る（設定で変更可能）
- 30秒応答がなければ不在着信

---

## 👶 未成年の通話制限

組織設定により、未成年メンバーの通話を制限可能。

| 設定 | 動作 |
|------|------|
| group_only | グループ通話のみ許可（1対1不可） |
| approved_only | 管理者が許可した相手のみ |
| none | 制限なし |

## 📊 通話履歴

| 表示情報 | 説明 |
|----------|------|
| 相手/グループ名 | 通話先 |
| 通話種別 | 音声/ビデオ |
| 日時 | 開始日時 |
| 時間 | 通話時間 |
| ステータス | 発信/着信/不在 |

## 🗄️ API

| エンドポイント | メソッド | 機能 |
|---------------|:--------:|------|
| `/api/calls.php?action=create` | POST | 通話開始 |
| `/api/calls.php?action=join` | POST | 通話参加 |
| `/api/calls.php?action=end` | POST | 通話終了 |
| `/api/calls.php?action=history` | GET | 通話履歴取得 |

### 通話開始レスポンス

{
  "success": true,

  "room_id": "social9_abc123",
  "join_url": "https://meet.jit.si/social9_abc123"

## 💡 実装ポイント

### Jitsi Meet埋め込み

```javascript
const domain = 'meet.jit.si';
const options = {
    roomName: 'social9_' + conversationId + '_' + Date.now(),
    width: '100%',
    height: '100%',
    parentNode: document.getElementById('jitsi-container'),
    userInfo: {
        displayName: userName
    },
    configOverwrite: {
        startWithAudioMuted: false,
        startWithVideoMuted: callType === 'audio'
    }
};
const api = new JitsiMeetExternalAPI(domain, options);
```
