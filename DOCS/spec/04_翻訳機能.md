# 翻訳機能

## 概要

多言語対応のためのメッセージ自動翻訳機能。
インターナショナルスクールや外国人とのコミュニケーションを支援します。

---

## 🌐 対応言語

| 言語 | コード | 優先度 |
|------|:------:|:------:|
| 🇯🇵 日本語 | ja | ⭐⭐⭐ |
| 🇺🇸 英語 | en | ⭐⭐⭐ |
| 🇨🇳 中国語（簡体） | zh-CN | ⭐⭐⭐ |
| 🇹🇼 中国語（繁体） | zh-TW | ⭐⭐ |
| 🇰🇷 韓国語 | ko | ⭐ |
| 🇻🇳 ベトナム語 | vi | ⭐ |

> 初期は日本語・英語・中国語の3言語を重点対応

---

## 🔧 翻訳の仕組み

### 1. 言語検出

メッセージ送信時に言語を自動検出。

```
「こんにちは」 → ja（日本語）
「Hello」     → en（英語）
「你好」      → zh-CN（中国語）
```

### 2. 翻訳実行

受信者の設定言語に合わせて翻訳を実行。

```
送信者：「明日の会議は10時からです」（日本語）
         ↓
受信者（英語設定）：「The meeting tomorrow starts at 10:00」
受信者（中国語設定）：「明天的会议从10点开始」
```

### 3. 翻訳表示

```
┌─────────────────────────────────────────────────────┐
│ John Smith                                  10:50   │
│                                                     │
│ Hello! Thank you for your help yesterday.           │
│                                                     │
│ ┌─────────────────────────────────────────────────┐ │
│ │ 🌐 翻訳（英語 → 日本語）                        │ │
│ │ こんにちは！昨日は助けていただきありがとう。    │ │
│ └─────────────────────────────────────────────────┘ │
│                                                     │
│ 👍                                                  │
└─────────────────────────────────────────────────────┘
```

## 🔘 翻訳操作

### 自動翻訳（ON）

- 送信者と異なる言語のメッセージを自動検出
- 翻訳を自動表示

### 手動翻訳

1. メッセージを長押し（またはホバー）
2. 「翻訳」をタップ
3. 翻訳先言語を選択
4. 翻訳結果を表示

## 🗄️ データベース設計

### message_translations テーブル

```sql
CREATE TABLE message_translations (
    id INT UNSIGNED AUTO_INCREMENT PRIMARY KEY,
    message_id INT UNSIGNED NOT NULL,
    
    source_language VARCHAR(10) NOT NULL,
    target_language VARCHAR(10) NOT NULL,
    translated_content TEXT NOT NULL,
    
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    
    FOREIGN KEY (message_id) REFERENCES messages(id) ON DELETE CASCADE,
    UNIQUE KEY unique_translation (message_id, target_language)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;
```

### user_language_settings テーブル

```sql
-- users テーブルに追加するカラム
```

## 💡 実装ポイント

### 翻訳キャッシュ

同じメッセージ・同じ言語の翻訳は再利用。

```php
// 既存の翻訳をチェック
$stmt = $pdo->prepare("
    SELECT translated_content FROM message_translations
    WHERE message_id = ? AND target_language = ?
");
$stmt->execute([$message_id, $target_lang]);
$cached = $stmt->fetch();

if ($cached) {
    return $cached['translated_content']; // キャッシュを返す
}

// なければAPI呼び出し
$translated = callTranslateAPI($content, $target_lang);
```

### Google Translate API呼び出し

```php
function translateText($text, $targetLang, $sourceLang = null) {
    $api_key = getenv('GOOGLE_TRANSLATE_API_KEY');
    
    $url = "https://translation.googleapis.com/language/translate/v2";
    $params = [
        'key' => $api_key,
        'q' => $text,
        'target' => $targetLang
    ];
    
    if ($sourceLang) {
        $params['source'] = $sourceLang;
    }
    
    $response = file_get_contents($url . '?' . http_build_query($params));
    $data = json_decode($response, true);
    
    return [
        'translated' => $data['data']['translations'][0]['translatedText'],
        'detected' => $data['data']['translations'][0]['detectedSourceLanguage'] ?? $sourceLang
    ];
}
```
