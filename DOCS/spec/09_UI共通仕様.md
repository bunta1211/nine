# UI共通仕様

## 概要

Social9のUI/UXにおける共通ルールを定義します。
ページ間の一貫性を保ち、ユーザーの違和感を軽減することを目的とします。

---

## 🎨 ページ遷移の共通ルール

### 基本方針

**ページ移動時の違和感を最小限にする**

ユーザーがページを移動した際に「別のアプリに来た」という感覚を与えないよう、以下のルールを守ります。

---

### 上パネル（ヘッダー）

すべてのページで共通の上パネルを使用します。

```
┌─────────────────────────────────────────────────────────────────┐
│ Social9  │  [ページタイトル]              [← チャットに戻る]    │
└─────────────────────────────────────────────────────────────────┘
```

| 要素 | 仕様 |
|------|------|
| **高さ** | 48px（`--header-height`） |
| **背景** | 緑グラデーション `linear-gradient(90deg, #4a6741 0%, #5a7a4a 50%, #6b8e50 100%)` |
| **ロゴ** | 「Social9」白・イタリック・20px |
| **区切り線** | 縦線（白半透明）、ロゴとタイトルの間 |
| **ページタイトル** | アイコン + タイトル（白・14px） |
| **戻るボタン** | 右端に配置、背景は白半透明 |

---

### パネル幅の統一

チャット画面と同じパネル幅を維持し、コンテンツエリアの位置を揃えます。

```
┌──────────────────────────────────────────────────────────────────┐
│                        上パネル（共通）                           │
├──────────┬────────────────────────────────────┬─────────────────┤
│  左余白   │           コンテンツエリア          │    右余白       │
│  260px   │              flex: 1               │    280px        │
└──────────┴────────────────────────────────────┴─────────────────┘
```

| パネル | 幅 | CSS変数 |
|--------|-----|---------|
| 左余白 | 260px | `--left-panel-width` |
| コンテンツ | 可変（残り幅） | `flex: 1` |
| 右余白 | 280px | `--right-panel-width` |

### レスポンシブ対応

| 画面幅 | 対応 |
|--------|------|
| 1024px以下 | 右余白を非表示 |
| 768px以下 | 左余白も非表示、コンテンツのみ表示 |

---

### 背景色

| 要素 | 色 |
|------|-----|
| body背景 | `#2d4a2d`（深緑） |
| 左右余白 | `rgba(255,255,255,0.95)` |
| コンテンツエリア | `rgba(255,255,255,0.98)` または白 |

---

## 📋 適用ページ一覧

| ページ | ファイル | 適用状況 |
|--------|----------|----------|
| チャット | `chat.php` | ベース（基準） |
| 設定 | `settings.php` | ✅ 適用済み |
| プロフィール | `profile.php` | 要対応 |
| Wish | `tasks.php` | 要対応 |

---

## 💡 新規ページ作成時のテンプレート

新しいページを作成する際は、以下の構造を使用してください。

### CSS変数

```css
:root {
    --header-height: 48px;
    --left-panel-width: 260px;
    --right-panel-width: 280px;
}
```

### HTML構造

```html
<!-- 上パネル -->
<header class="top-panel">
    <div class="logo">Social9</div>
    <div class="header-divider"></div>
    <span class="page-title">🔧 ページタイトル</span>
    <a href="chat.php" class="back-btn">← チャットに戻る</a>
</header>

<!-- メインコンテナ -->
<div class="main-container">
    <aside class="left-spacer"></aside>
    <main class="center-panel">
        <!-- ページ固有のコンテンツ -->
    </main>
    <aside class="right-spacer"></aside>
</div>
```
