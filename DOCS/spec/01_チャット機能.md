# チャット機能

## 概要

Social9の基本機能。テキスト・画像・ファイルの送受信を行います。

---

## 💬 メッセージの種類

| 種類 | 説明 | 対応形式 |
|------|------|----------|
| テキスト | 通常のメッセージ | プレーンテキスト、URL自動リンク |
| 画像 | 画像ファイル | JPEG, PNG, GIF, WebP |
| ファイル | 添付ファイル | PDF, Word, Excel, ZIP 等 |
| システム | 自動生成 | 参加/退出通知など |

## 🗄️ データベース

### message_mentions（メンション検索用）

メンション（@ユーザー名）は検索性を重視して別テーブルで管理。

```sql
CREATE TABLE message_mentions (
    id INT UNSIGNED AUTO_INCREMENT PRIMARY KEY,
    message_id INT UNSIGNED NOT NULL,
    mentioned_user_id INT UNSIGNED NOT NULL,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    
    UNIQUE KEY unique_mention (message_id, mentioned_user_id),
    INDEX idx_mentioned_user (mentioned_user_id, created_at DESC)
);
```

### メンション検索クエリ例

```sql
-- 自分がメンションされたメッセージ一覧
SELECT 
    m.*,
    c.name AS conversation_name,
    u.display_name AS sender_name
FROM message_mentions mm
INNER JOIN messages m ON m.id = mm.message_id
INNER JOIN conversations c ON c.id = m.conversation_id
INNER JOIN users u ON u.id = m.sender_id
WHERE mm.mentioned_user_id = :user_id
ORDER BY mm.created_at DESC
LIMIT 50;
```

## 💡 実装ポイント

### メッセージ送信時のメンション抽出

```php
function extractAndSaveMentions($pdo, $messageId, $content) {
    // @ユーザー名 を抽出
    preg_match_all('/@([a-zA-Z0-9_\p{L}]+)/u', $content, $matches);
    
    if (empty($matches[1])) {
        return [];
    }
    
    $mentionedUsers = [];
    $stmt = $pdo->prepare("SELECT id, display_name FROM users WHERE display_name = ?");
    
    foreach (array_unique($matches[1]) as $name) {
        $stmt->execute([$name]);
        $user = $stmt->fetch();
        
        if ($user) {
            $mentionedUsers[] = $user;
            
            // message_mentionsに保存
            $pdo->prepare("
                INSERT IGNORE INTO message_mentions (message_id, mentioned_user_id)
                VALUES (?, ?)
            ")->execute([$messageId, $user['id']]);
        }
    }
    
    return $mentionedUsers;
}
```

### 効率的なメッセージ取得（N+1回避）

```php
function getMessages($pdo, $conversationId, $limit = 50, $beforeId = null) {
    $sql = "
        SELECT 
            m.id,
            m.content,
            m.message_type,
            m.reply_to_id,
            m.is_edited,
            m.is_pinned,
            m.created_at,
            m.sender_id,
            u.display_name AS sender_name,
            u.avatar_path AS sender_avatar,
            (SELECT GROUP_CONCAT(CONCAT(reaction_type, ':', COUNT(*)) SEPARATOR ',')
             FROM message_reactions 
             WHERE message_id = m.id 
             GROUP BY reaction_type) AS reactions,
            (SELECT COUNT(*) FROM message_mentions WHERE message_id = m.id) AS mention_count
        FROM messages m
        INNER JOIN users u ON u.id = m.sender_id
        WHERE m.conversation_id = :conversation_id
        AND m.deleted_at IS NULL
    ";
    
    if ($beforeId) {
        $sql .= " AND m.id < :before_id";
    }
    
    $sql .= " ORDER BY m.created_at DESC LIMIT :limit";
    
    $stmt = $pdo->prepare($sql);
    $stmt->bindValue(':conversation_id', $conversationId, PDO::PARAM_INT);
    $stmt->bindValue(':limit', $limit, PDO::PARAM_INT);
    if ($beforeId) {
        $stmt->bindValue(':before_id', $beforeId, PDO::PARAM_INT);
    }
    $stmt->execute();
    
    return $stmt->fetchAll();
}
```
