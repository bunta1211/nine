# プッシュ通知 セットアップガイド

携帯の通知画面にSocial9の通知を表示するには、以下の手順が必要です。

## 1. Composerでライブラリをインストール

Web Pushプロトコルでは、ペイロードのAES128GCM暗号化が必須です。
`minishlink/web-push` ライブラリを使用してください。

```bash
# プロジェクトルートで実行
composer install
```

Composerが未インストールの場合は、[getcomposer.org](https://getcomposer.org/) からインストールしてください。

## 2. 通知許可の有効化

1. チャット画面（`chat.php`）にアクセス
2. 「通知を有効にしますか？」バナーが表示されたら「有効にする」をタップ
3. ブラウザの通知許可ダイアログで「許可」を選択

## 3. テスト通知の送信

1. 設定画面の「プッシュ通知」セクションで「テスト通知を送信」をタップ
2. または、API経由で `api/push.php?action=test` をPOST

正常に動作している場合、デバイスの通知リストに Social9 の通知が表示されます。

## トラブルシューティング

### 通知が届かない場合

- **HTTPS必須**: プッシュ通知は `https://` または `localhost` で動作します
- **通知許可**: ブラウザ設定で social9.jp の通知が「許可」になっているか確認
- **ライブラリ未インストール**: `vendor/` フォルダが存在するか確認。存在しない場合は `composer install` を実行

### エラーログの確認

`logs/` ディレクトリまたはPHPのエラーログに以下が出る場合：

```
Push: minishlink/web-push がインストールされていません
```

→ `composer install` を実行してください。
