# 着信音テスト手順

着信音（メッセージ・通話）の設定確認とテスト方法です。

## 設定画面でのテスト

1. **設定 → 着信音** を開く
2. **着信音（自分宛メッセージ）**
   - 秒数ボタン（1秒/3秒/5秒）をクリックで試聴・選択
   - 「🔔 着信音テスト（選択中の音を再生）」で現在選択中の音を再生
   - 「メッセージ着信音をテスト」ボタンでも同様にテスト可能
3. **着信音（通話）**
   - 秒数ボタンで試聴・選択
   - 「通話着信音をテスト」ボタンでテスト
4. **重要**: 変更後は **「設定を保存」** を必ずクリックしないと保存されません

## メッセージ受信時の着信音確認

1. チャットを開いた状態で、別の端末または別アカウントからメッセージを送信
2. 着信音が鳴る条件：
   - **パソコン版**: 「自分宛のみ」のときは、自分宛メンションまたはDMのみ鳴る
   - **携帯版**: 同様にトリガー設定に依存
3. 音が鳴らない場合の確認：
   - ブラウザがミュートになっていないか
   - 着信音が「サイレント」になっていないか
   - ページを1回クリック/タッチしてオーディオコンテキストを有効化する（Chrome等はユーザー操作が必要）

## トラブルシューティング

| 症状 | 確認事項 |
|------|----------|
| 設定が保存されない | 「設定を保存」ボタンをクリックしたか確認。秒数ボタンを選んだだけでは保存されません |
| チャットで古い音が鳴る | 設定保存後、チャットタブを開いたままの場合は次のメッセージから新しい音が適用されます（BroadcastChannelで自動更新） |

## 技術メモ

- チャットは `BroadcastChannel('social9-settings')` で `ringtone_saved` を受け取り、設定キャッシュを無効化する
- 着信音保存後、チャットタブを開いたままでも次のメッセージから新しい着信音が適用される
