## 🚀 投資管理スキーム概要
1. **投資家登録**: 資産1億円以上・投資経験1年以上を確認（セルフデクレア方式）
2. **反社チェック**: 投資実行前に外部APIにより自動照会
3. **資金調達**: 4ヶ月ごとに最大9,900万円を募集（システムによる上限制御）
4. **会計自動処理**: 出資額の1/2を「資本金」、残りの1/2を「資本準備金」として名簿に自動記録
5. **株主名簿管理**: アプリ内機能として名簿を保持し、二次流通（取引場）は提供しない
