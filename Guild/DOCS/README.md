# Guild - 報酬分配システム

## 概要

Guildは保育園・学童保育の運営における報酬分配を管理するシステムです。
「Earth」という仮想通貨を使用して、依頼の報酬や感謝の気持ちをやり取りします。

## 主な機能

### 依頼システム
- **公開依頼**: 掲示板形式で誰でも立候補可能
- **指名依頼**: 特定の人への依頼（断れる）
- **業務指令**: 断れない指示（高Earth）
- **勤務交代依頼**: シフト交代のお願い
- **個人依頼**: 自分のEarthを使った依頼
- **感謝の気持ち**: チップ的な送付（匿名可）
- **特別報酬**: 期待以上の仕事への追加報酬

### Earth管理
- 1 Earth = 10円
- 年度初めに在籍年数・役職ボーナスを配布
- 前借り申請（保有の80%まで）
- 3月10日で年度末精算

### 共有カレンダー
- 勤務予定・休日の登録
- ギルド3役は担当メンバーの予定を確認可能

### 余力表示
- 本日・今週・今月・来月以降の余力を設定
- 新規依頼不可期間の設定

## 技術仕様

- **言語**: PHP 7.4+
- **データベース**: MySQL 5.7+ / MariaDB 10.3+
- **対応言語**: 日本語、英語、中国語
- **対応端末**: PC、スマートフォン、タブレット

## インストール

### 1. データベースセットアップ

```sql
-- database/schema.sql を実行
mysql -u root -p social9 < database/schema.sql
```

### 2. 設定確認

`config/database.php` でデータベース接続設定を確認してください。
Social9と同じデータベースを使用します。

### 3. アクセス

```
http://localhost/nine/Guild/
```

## 役割と権限

### システム権限
- **システム管理者**: 全機能にアクセス可能
- **給与支払い担当**: 支払い管理のみ

### ギルド権限
- **ギルドリーダー**: 予算管理、全依頼発行、メンバー管理
- **サブリーダー**: 依頼発行（業務指令含む）、勤務交代承認
- **コーディネーター**: ギルド依頼発行、メンバー管理
- **メンバー**: 依頼応募、個人依頼

## 支払いスケジュール

| 活動期間 | 支給日 |
|---------|--------|
| 4月〜6月 | 8月20日 |
| 7月〜9月 | 9月末 |
| 10月〜12月 | 3月末（前借り可） |
| 1月〜3月10日 | 3月末 |
