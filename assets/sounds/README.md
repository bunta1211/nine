# 効果音・着信音用音源

ボタン押下・送信完了・着信など、UIで再生する音声ファイルをこのフォルダに配置します。

## 着信音としての利用

- **自分宛メッセージ着信音**: 設定画面「着信音」で選択した音が、チャットで自分宛メッセージを受信したときに1回再生されます。
- **通話着信音**: 設定画面「着信音（通話）」で選択した音が、音声/ビデオ通話の着信時にループ再生され、応答または拒否で停止します。
- **試聴**: 設定画面の「試聴」ボタンで、選択中の音を1回再生して確認できます（秒数指定は廃止し、ファイルを1回再生する方式です）。

選択肢は `config/ringtone_sounds.php` の `$RINGTONE_SOUNDS_LIST` で定義されています。ファイルを追加した場合は同設定に id と表示名を追加してください。

## 推奨

- **形式**: MP3 または WAV（ブラウザ互換のため MP3 を推奨）
- ファイル名は半角英数字＋アンダースコアにすると、どの環境でも安全です。

実装側では `assets/sounds/〇〇.mp3` のようなパスで読み込みます。
